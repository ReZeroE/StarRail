To run the game, the 'starrail' module needs to know where the game Honkai Star Rail is located.
Below, please provide the path to an official Windows installation of Honkai Star Rail.

    Example paths:
        D:\
        D:\Honkai Star Rail
        D:\Honkai Star Rail\Star Rail
        D:\Honkai Star Rail\Star Rail\Game
        D:\Honkai Star Rail\Star Rail\Game\StarRail.exe

    ALL OF THE PATHS ABOVE ARE FINE. The module will automatically resolve the rest.
    Note: This will overwrite the previously set path.
